How to install it ?

The following dependancies are required :

  • BEDTools (version 2.21.0)
  • vcf2diploid (version 0.2.6a)
  • samtools (version )
  • ART (version )

To install :

tar -zxvf master.tar.gz
cd pip-master
make CONFIG_SYS=config-install.txt install
